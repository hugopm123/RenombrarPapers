# 📚 PDF Paper Renamer (Universal)

Herramienta profesional para automatizar el renombrado de bibliotecas científicas mediante **Análisis Jerárquico Visual**.

## 🛠️ Modos de Uso

### 1. Interfaz Web (Recomendado)
Para una experiencia visual e intuitiva, utiliza la aplicación web local.
```bash
pip install streamlit pymupdf
streamlit run app.py
```

### 2. Terminal (CLI)
Para uso rápido desde la línea de comandos:
```bash
pip install pymupdf
python3 rename_papers.py "/ruta/a/tus/papers"
```

## ✨ Características
- **Motor Visual 4.2**: Detecta títulos basándose en el tamaño de la fuente.
- **Filtros Inteligentes**: Ignora preprints, universidades y metadatos.
- **Seguridad**: Permite previsualizar los cambios antes de aplicarlos.
- **Universal**: Compatible con papers en inglés y español.
